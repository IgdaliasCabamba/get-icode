Enjoy IntelliSense.

Go beyond syntax highlighting and autocompletion with IntelliSense, which provides smart completions, hints, and edit-time parsing based on variable types, function definitions, and imported modules!

Extensible and customizable.

Get even more features by installing extensions to add new languages, themes, debuggers and to connect to additional services. Extensions run in separate processes, ensuring they don't slow down your editor. Learn more about extensions!

Fast and light.

Icode has Qt as the basis for its graphical interface and QScintilla for the editor so it's quite light for the amount of features it offers, and compared to some other editors! And a good codebase was used, with features like threads, to prevent extensions or a weak computer from interfering with your productivity!

More Features

Integrated terminal

Icode has an integrated cross-platform terminal that emulates the native terminal giving you more productivity and eliminating the need to open multiple windows

Source Control

Create repositories or clone the web! View and manage your commits and repositories, check status, log in, add or remove files, and track file status while editing!

Run and Debug

Run and debug, run your code from Icode, use an extension to debug and analyze your code!

Go To

Go to any line, part or file in your editor with powerful goto functions!

Cross Platform

Icode was made from scratch using PyQt(Qt) so it can run on any platform supported by Qt, while official support is for Linux and Windows, I will support MacOS soon.

Highly customizable

Qt supports palettes and stylesheets, so your imagination is the limit when it comes to customization, adjust Icode to your tastes, finding a style or creating one!

Extensible

Icode follows the idea of many modern editors, a simple thing that can be extended! Find extensions that increase your productivity here!

April Bot

April is one of Icode's differentiators and an assistant for you, April is there to help you with things like configuring icode, getting answers to questions, analyzing your code and much more!
